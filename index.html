FAQ

Common questions

Is my biodata stored anywhere?
No. All data is processed entirely in your browser. Nothing is sent to any server. We do not have a backend at all — this is a static website hosted on GitHub Pages.
Is this actually free?
Yes, 100% free forever. No subscription, no premium plan, no watermark on downloads. All three templates and all features are available to everyone without any cost.
Does it support Hindu, Muslim, and Christian formats?
Yes. Select your religion in the Family & Religion section. For Hindu profiles, fields like Gotra, Rashi, Nakshatra and Manglik status appear automatically. Muslim and Christian profiles get community/denomination fields instead.
Can I use my own photo?
Yes. Use the Photo section to upload a photo. It is read using browser's FileReader API and only used to render the preview and PDF. It is never transmitted anywhere.
How good is the PDF quality?
We use html2canvas at 3× scale to capture the preview, then embed it in a proper A4 PDF via jsPDF. The result is print-ready at high resolution — ideal for WhatsApp sharing, email attachments, or printing.
